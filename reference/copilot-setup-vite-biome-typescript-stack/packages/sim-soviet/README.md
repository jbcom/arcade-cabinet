# Sim Soviet 3000

A command-table city management simulation about quotas, utilities, and austere planning.

## Creative Pillars

- Command-table planning: the city should feel like a physical model under review.
- Quota pressure: progress is always visible and politically loaded.
- Utilities as visible systems: power, housing, industry, and services should be distinguishable.

## Presentation Direction

Cold slate terrain, red/orange civic accents, and a fixed isometric board make political pressure and infrastructure state visible. Buildings stay simple so the player can reason about the plan quickly.

## Responsive and Android Contract

- The root fills its parent with `GameViewport`.
- The board stays in camera target across desktop and mobile.
- HUD and building palette occupy predictable edges.
- The package has a standalone Vite/Capacitor app shell.
